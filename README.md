# acidtide

**acidtide** is an interactive, browser-based ocean simulation that visualizes the effects of ocean acidification over time. watch how pH changes impact coral, while plankton float and bubbles rise through soft, flowing S-shaped waves.

## Features

- **Coral health indicator** that changes based on pH  
- **Interactive timeline**: change the year to see pH trends  

Built with p5.js
